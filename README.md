# EnsnarlNet
This repository is to maintain the code for the following paper
"Yu Tian, Chinmayi Subramanya and Carl D. Modes, [*GEMINI: Generalized Ensnarlment Measure from Incomplete-linkage of Network-network Interactions*](), arXiv.". 


## Requirement
The code is written in Python, and the following packages are required. 

* [NumPy](https://github.com/numpy/numpy) = 2.0.2 (Matrix Manipulation)
* [NetworkX](https://github.com/networkx/networkx) = 3.6.1 (Based Graph library)
* [Matplotlib](https://github.com/matplotlib/matplotlib) = 3.10.0 (Visulisation)
* [Plotly](https://github.com/plotly/plotly.py) = 5.24.1 (3D Visualization)
* [Kaleido](https://github.com/plotly/Kaleido) = 0.2.1 (3D Visualization Static Figures Generation)

## Code
The following files are contained under the code folder. 

- ensnarlNet_lattice_all.ipynb: GEMINI analysis on ensnarled ladder and step lattices.
- ensnarlNet_data_all.ipynb: GEMINI analysis on real data of mouse brain vasculature.
